# Trader Behavior vs Market Sentiment Analysis

## Overview
This project analyzes how Bitcoin market sentiment impacts trader performance using historical Hyperliquid trade data and the Fear & Greed Index.

## Key Insights
- Profitability and win rates increase during Greed and Extreme Greed phases.
- Fear markets reward contrarian BUY strategies.
- SELL strategies dominate during Greed and Extreme Greed.
- Higher profits in Extreme Greed are driven by market momentum rather than larger trade sizes.
